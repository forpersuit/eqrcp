## 常见问题

### Q1: Fork的项目能申请软著吗？
**A**: 可以。只要您进行了实质性的独立开发和改进，就可以申请。您的项目已经使用独立的模块名、添加了大量原创功能，完全符合申请条件。

### Q2: 需要提交全部源代码吗？
**A**: 不需要。只需提交前30页+后30页共60页源代码即可。

### Q3: 开源软件能申请软著吗？
**A**: 可以。开源不影响著作权归属，您仍然是著作权人。

### Q4: 申请表中的日期如何填写？
**A**: 
- 开发完成日期：软件基本功能完成的日期
- 首次发表日期：如果未公开发布选"未发表"，如果已发布填写发布日期
- 首次发表日期必须晚于或等于开发完成日期

### Q5: 审查过程中可能遇到什么问题？
**A**: 
- 材料不全：按补正通知补充即可
- 代码格式问题：确保每页50行以上
- 信息不一致：确保所有文档中的信息统一
