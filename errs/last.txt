  Q3：C3 代码签名要什么？

  macOS 端

  资质：
  - Apple Developer Program 会员：$99/年
  - 在 Apple 网站申请「Developer ID Application」证书，下载到 Keychain
  - 需要 Apple ID + App-Specific Password（在 appleid.apple.com 生成，免费）

  流程（已经文档化在 docs/wails-build-issue.md 里）：
  codesign --deep --force --options runtime --sign "Developer ID Application: <Your Name>" eqrcp-desktop.app
  create-dmg eqrcp-desktop.app
  xcrun notarytool submit eqrcp-desktop.dmg --apple-id $APPLE_ID --team-id $TEAM_ID --password $APP_PASSWORD
  --wait
  xcrun stapler staple eqrcp-desktop.dmg

  CI Secrets 要存：
  - APPLE_DEVELOPER_CERT_P12_BASE64（证书导出为 .p12 后 base64）
  - APPLE_DEVELOPER_CERT_PASSWORD
  - APPLE_TEAM_ID（10 字符）
  - APPLE_ID（邮箱）
  - APPLE_ID_PASSWORD（app-specific）

  Windows 端

  两条路径，建议第二条：

  ┌────────────────────────┬──────────────────────┬────────────────────────────────────────────────────────┐
  │          方案          │         费用         │                          优劣                          │
  ├────────────────────────┼──────────────────────┼────────────────────────────────────────────────────────┤
  │                        │ OV 约                │ EV 立即获得 SmartScreen                                │
  │ 买 OV/EV 代码签名证书  │ $200-300/年，EV 约   │ 信誉，用户下载不弹"未识别发行者"警告                   │
  │                        │ $400-700/年          │                                                        │
  ├────────────────────────┼──────────────────────┼────────────────────────────────────────────────────────┤
  │ Azure Trusted          │ $9.99/月 = $120/年   │ Microsoft 自家服务，2024 年正式上线。无硬件 token，CI  │
  │ Signing（推荐个人）    │                      │ 友好。信誉建立较慢 (大约首批用户量) 但起步成本最低     │
  └────────────────────────┴──────────────────────┴────────────────────────────────────────────────────────┘

  流程（OV/EV cert）：

  Q3 续：Windows 上未签名 .exe 的现实
