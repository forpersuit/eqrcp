## Usage

```bash
# Start desktop agent
eqrcp desktop agent-start -B

# Start chat session
eqrcp desktop chat

# Check agent status
eqrcp desktop agent-status

# Open agent status page
eqrcp desktop agent-open

# Stop current chat
eqrcp desktop agent-stop-current
```

## Architecture

Chat sessions now follow the same lifecycle as share/receive tasks:
- State tracking: waiting → active → ended
- Task lifecycle: running → completed
- History preservation with session details
- Support for stop and repeat operations
